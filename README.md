# DataStructure_projec

제공된 데이터를 이용하여 주어진 문제를 해결한다.
이때, 가장 합리적인 자료구조를 이용하여 문제를 해결한다.


- 제공된 데이터

------ | -----| -------------- | ----- | -------- | ------- | -- | -- | --- | --- | ---- | -- | ---------------- | -- | ---- | ------------ | ----------------
유연수 |	홍지동 |	010-2019-1711 |	유정인 |	95.03.15 |	6개월권 |	N |	- |	27.5 |	25 |	20 |	1 |	2019년 05월 01일 |	25 |	0 | 2019년 06월 30일 |	2019년 07월 31일
